# Design System
* `/packages/*` 디렉토리는 각각의 npm 프로젝트이고, 이를 중앙에서 관리할수 있도록 **Monorepo** 아키텍처를 선택했습니다.
* `lerna` 라이브러리로 프로젝트를 관리합니다.
